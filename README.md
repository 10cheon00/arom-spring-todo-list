# Git Convention

| message  | description    |
|----------|----------------|
| feat     | 새로운 기능 추가      |
| fix      | 문제 해결 또는 오류 수정 |
| rename   | 파일/폴더 이름 변경    |
| refactor | 코드 리팩토링        |
| chore    | 빌드, 패키지 매니징 변경 |
| test     | 테스트 코드 추가, 변경 |
